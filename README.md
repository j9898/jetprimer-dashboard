# JetPrimer Dashboard

미국 LLC 설립 서비스를 위한 고객 대시보드.

## 기술 스택

- **프레임워크**: Next.js 16 (App Router)
- **스타일링**: Tailwind CSS v4
- **인증**: Supabase Auth (Google OAuth + Magic Link)
- **배포**: Vercel

## 개발 서버 실행

```bash
cd jetprimer-next
npm install
npm run dev
```

http://localhost:3000 에서 확인

## 배포

GitHub에 푸시하면 Vercel에서 자동 배포됩니다.

- **프로덕션**: https://dashboard.jetprimer.com
